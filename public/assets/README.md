# Logo Placement Instructions

Please save the NGO logo as `logo.png` in this directory.

The logo should be:
- Transparent background PNG format
- Recommended size: 512x512px or similar square dimensions
- The logo will be displayed at 48-64px in the interface

Once you place the logo file here, it will automatically appear in:
- Header (top navigation)
- Footer
- All pages

Current expected path: `/public/assets/logo.png`
